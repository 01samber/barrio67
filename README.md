# Barrio 67 — Bohemian Resto Pub

A modern, responsive, SEO-optimized website for Barrio 67, a bohemian resto-pub in Batroun Old Souk, Lebanon.

## Features

- **Responsive Design** — Mobile-first, works on all devices
- **SEO Optimized** — Meta tags, semantic HTML5, Open Graph
- **Performance** — Lazy loading images, optimized asset loading
- **Accessibility** — ARIA labels, reduced-motion support

## Structure

```
Barrio 67/
├── index.html          # Main page
├── css/
│   └── styles.css     # All styles
├── js/
│   └── script.js      # Navigation & animations
└── README.md
```

## Setup

1. Open `index.html` in a browser, or
2. Serve via a local server (e.g. `npx serve .`)

## Customization

- **Images**: Replace Unsplash URLs in `index.html` with your own photos
- **Google Maps**: Update the iframe `src` in the Visit section with your exact location embed
- **Social Links**: Update Instagram/Facebook URLs to your actual profiles
- **Menu**: Edit menu items and prices in the Menu Preview section
- **Phone**: Format is `+96170211299` (Lebanon)

## Browser Support

Modern browsers (Chrome, Firefox, Safari, Edge). Smooth scroll and CSS Grid used.
